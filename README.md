⚡ Installation
Clone Repository
git clone https://github.com/YOUR_USERNAME/careerverse-ai.git
cd careerverse-ai
📦 Install Dependencies
npm install
▶️ Run Frontend
npm run dev
▶️ Run Backend
npm run dev:api
▶️ Run Full Stack Together
npm run dev:all
🌐 Environment Variables

Create .env files.

Frontend
VITE_API_URL=http://localhost:5000
Backend
PORT=5000
JWT_SECRET=your_secret_key
